Kontent.ai Custom App - Iframe Test

This page tests if your custom app can be embedded in an iframe.

✅ If you can see the app below, iframe embedding is working correctly!
❌ If you see a blank iframe or an error, there's an iframe embedding issue.

Troubleshooting:

  • Make sure your app is running on http://localhost:3000
  • Check that the Content-Security-Policy headers are set correctly
  • Ensure X-Frame-Options is not set to 'deny'
